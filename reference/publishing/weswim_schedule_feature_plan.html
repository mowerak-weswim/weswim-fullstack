진행 중

확정 완료

작성자가 한 항목 확정. 카드가 일정 공지로 전환. 전체 멤버 알림 발송.

우측 "다음 일정" 위젯

가장 가까운 일정 1개 프리뷰. 투표 중이면 투표 현황(N명 참여), 확정이면 날짜·장소 표시. "전체 일정 보기 →" 클릭 시 공지·벙개 탭으로 이동.

공지·벙개 탭 피드

일정 카드가 상단 고정. 투표 카드·공지 카드 인라인 렌더링. 일반 게시글 아래로 정렬. 완료 일정은 "지난 일정" 섹션으로 분리.

투표 기능은 공지·벙개 탭에서만 생성 가능. 잡담탭·에티켓탭에서는 일반 게시글 작성만 허용 — 탭 혼선 방지.
